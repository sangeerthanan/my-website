## 🆘 Having Issues?

**Port 3000 already in use?**
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Mac/Linux
lsof -ti:3000 | xargs kill -9
```

**Styles not updating?**
```bash
rm -rf .next
npm run dev
```

**Can't install dependencies?**
```bash
npm cache clean --force
rm -rf node_modules
npm install
```

Need more help? Check the relevant `.md` file for your issue.

## 🎓 Learning Path

If you want to extend this:

1. **Add New Section**
   - Create file in `sections/`
   - Import in `app/page.tsx`
   - Add styles and animations

2. **Add New Component**
   - Create file in `components/`
   - Export and import where needed

3. **Change Colors**
   - Edit `tailwind.config.ts`
   - All components auto-update

4. **Add 3D Effects**
   - Three.js is already installed
   - Check components for examples
